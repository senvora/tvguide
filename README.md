# 📺 TV Guide (EPG)
[![Tempest EPG Grabber](https://github.com/senvora/tvguide/actions/workflows/tempest-epg-grabber.yml/badge.svg)](https://github.com/senvora/tvguide/actions/workflows/tempest-epg-grabber.yml)

This repository provides an automatically updated **Electronic Program Guide (EPG)** for IPTV playlists.  
It is designed to fetch, process, and generate EPG XML files that can be used in media players, IPTV apps, and streaming services.

---

## 🚀 Features
- Automatic EPG updates via GitHub Actions
- Optimized to run on a schedule (every 2 hours / daily updates)
- Supports multiple channels with proper metadata
- Clean and simple XMLTV output format
